# Website Desa Persiapan Kiera

Website resmi Desa Persiapan Kiera, Kecamatan Moa, Kabupaten Maluku Barat Daya, Maluku.

## 🌐 Website Online
[Buka Website](https://buatwebsite123.github.io/desa-cantik-kiera-github/)

## 🔄 Cara Update Data Website

1. Jalankan server lokal: `node server.js`
2. Login ke admin panel: `http://localhost:3000/admin`
3. Update data (berita, galeri, pengumuman, dll)
4. Export data ke JSON: `node export-data.js`
5. Push ke GitHub:
   ```
   git add .
   git commit -m "Update data"
   git push
   ```
6. Website online otomatis terupdate!
